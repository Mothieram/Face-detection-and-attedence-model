# Attendance cooldown store
#
# Controls how long a person is blocked from re-scanning after

#
# Redis makes the cooldown work across multiple workers or servers.
# Falls back to in-process dict automatically if Redis is unavailable
# (single-worker mode — safe for development).
#
# To enable Redis on Windows: install Memurai for Redis
#   https://www.memurai.com/get-memurai
#   Then: pip install redis>=5.0.0
#
# To enable Redis on Linux:
#   sudo apt install redis-server
#   pip install redis>=5.0.0
# ------------------------------------------------------------
